                                               Paketli Gıda Analizi Mobil Uygulaması

Paketli gıda analizi uygulaması ile  paketli gıda satın almadan önce ürünlerin içerik bilgisi, içerdiği katkı maddeleri ve, sağlığa etkileri, alerjen madde barındıran ürünlerin kontrolü, bu ürünlerin kiloya etkileri sorgulatılarak, tüketiciye sunmak amaçlanmıştır. Bu çalışma sayesinde tüketici daha satın almadan önce mobil cihazının kamerasını kullanıp ürün üzerindeki barkodu taratarak veya manuel şekilde barkod bilgisi girerek satın aldığı herhangi bir ürünün detaylı bilgilerini inceleyebilecektir.
